| `full_name` | Text | Display name |
| `follower_count` | Integer | Number of followers |
| `following_count` | Integer | Number of accounts following |
| `tweet_count` | Integer | Total tweets posted |
| `is_verified` | Boolean | Verification badge status (blue checkmark) |
| `biography` | Text | Profile bio text |
| `location` | Text | User-provided location |
| `website` | Text | Profile website URL |
| `detected_gender` | Text | Gender detection result |
| `account_created` | Date | Account creation date |

## Troubleshooting

<AccordionGroup>
  <Accordion title="X API Authentication Failed">
    **Possible Causes:**
    - API credentials expired or invalid
    - Account suspended or restricted
    - API access tier insufficient
    
    **Solutions:**
    - Verify API keys in environment variables
    - Check X Developer Portal for account status
    - Ensure proper API tier subscription (Free, Basic, Pro)
    - Regenerate API tokens if expired
  </Accordion>
  
  <Accordion title="Rate Limit Exceeded">
    **Possible Causes:**
    - Too many requests in 15-minute window
    - Insufficient delay between scraping sessions
    - Multiple concurrent scraping jobs
    
    **Solutions:**
    - Wait for rate limit reset (15 minutes)
    - Implement exponential backoff
    - Reduce scraping frequency
    - Upgrade API tier for higher limits
  </Accordion>
  
  <Accordion title="Follower List Incomplete">
    **Possible Causes:**
    - Account has too many followers (API pagination limits)
    - Some followers are private or restricted
    - API returning partial results
    
    **Solutions:**
    - Implement pagination handling for large follower lists
    - Accept that some accounts may not be accessible
    - Focus on most recent or most engaged followers
  </Accordion>
  
  <Accordion title="Low-Quality Profile Data">
    **Possible Causes:**
    - Many bot or spam accounts in follower list
    - Inactive or abandoned accounts
    - Minimal profile information provided by users
    
    **Solutions:**
    - Filter by account age (exclude accounts created recently)
    - Check tweet count (exclude accounts with less than 10 tweets)
    - Verify follower/following ratio for authenticity
    - Use bio keywords to filter for relevant professionals
  </Accordion>
</AccordionGroup>
