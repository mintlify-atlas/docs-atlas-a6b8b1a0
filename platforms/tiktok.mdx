### Data Fields Extracted

TikTok scraping collects comprehensive creator profile data:

| Field | Type | Description |
|-------|------|-------------|
| `id` | Text | TikTok user ID |
| `username` | Text | TikTok handle (without @) |
| `full_name` | Text | Display name |
| `follower_count` | Integer | Number of followers |
| `following_count` | Integer | Number of accounts following |
| `likes_count` | Integer | Total likes across all videos |
| `video_count` | Integer | Total videos posted |
| `is_verified` | Boolean | Verification badge status |
| `biography` | Text | Profile bio text |
| `detected_gender` | Text | Gender detection result |

## Troubleshooting

<AccordionGroup>
  <Accordion title="TikTok Scraping Blocked">
    **Possible Causes:**
    - Rate limits exceeded
    - IP address temporarily blocked
    - Account flagged for suspicious activity
    
    **Solutions:**
    - Wait 24-48 hours before retrying
    - Reduce scraping frequency
    - Use residential proxy services
    - Rotate scraping accounts
  </Accordion>
  
  <Accordion title="Follower Data Incomplete">
    **Possible Causes:**
    - TikTok API limitations
    - Private accounts blocking data access
    - Recent platform changes
    
    **Solutions:**
    - Focus on public creator accounts
    - Accept that TikTok data may be less detailed than Instagram
    - Prioritize engagement metrics over follower lists
  </Accordion>
  
  <Accordion title="Low Gender Detection Accuracy">
    **Possible Causes:**
    - TikTok usernames often don't include real names
    - Display names may be nicknames or brand names
    - International users with non-Western names
    
    **Solutions:**
    - Expect lower accuracy (~70%) compared to Instagram
    - Use bio text analysis as supplementary signal
    - Consider manual review of high-value segments
  </Accordion>
  
  <Accordion title="Regional Content Mismatch">
    **Possible Causes:**
    - TikTok's algorithm shows different content by region
    - Creator audiences concentrated in specific countries
    
    **Solutions:**
    - Verify creator's primary audience geography
    - Use location-specific hashtags for targeting
    - Consider using VPN to access region-specific content
  </Accordion>
</AccordionGroup>
