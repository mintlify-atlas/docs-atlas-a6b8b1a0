## Overview

Threads is Meta's text-based conversation platform, closely integrated with Instagram. The Click Creators Scraper Client includes Threads support for expanding social media outreach beyond traditional Instagram engagement.

<Info>
Threads integration is currently active and available for use, leveraging similar scraping infrastructure to Instagram.
</Info>

### Data Fields Extracted

Threads scraping collects similar profile data to Instagram:

| Field | Type | Description |
|-------|------|-------------|
| `id` | Text | Threads profile ID |
| `username` | Text | Threads handle (often matches Instagram) |
| `full_name` | Text | Display name |
| `follower_count` | Integer | Number of followers |
| `following_count` | Integer | Number of accounts following |
| `is_verified` | Boolean | Verification badge status |
| `biography` | Text | Profile bio text |
| `instagram_username` | Text | Linked Instagram account (if available) |
| `detected_gender` | Text | Gender detection result |

### Creating a Threads Scraping Job


## API Integration

### Threads-Specific Endpoints

Threads uses the same API endpoints as Instagram with platform-specific parameters:

```bash
POST /api/scrape-followers
{
  "accounts": ["threads_username"],
  "platform": "threads",
  "targetGender": "male"
}
```

<Note>
API responses include Threads-specific metadata and cross-platform linking information.
</Note>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Threads Profile Not Found">
    **Possible Causes:**
    - User hasn't created Threads account despite having Instagram
    - Username differs between Instagram and Threads
    - Profile is private or restricted
    
    **Solutions:**
    - Verify account exists on Threads app
    - Check if username matches Instagram handle
    - Ensure source account is public
  </Accordion>
  
  <Accordion title="Limited Profile Data">
    **Possible Causes:**
    - Threads API provides less data than Instagram
    - New accounts with minimal information
    - Privacy settings limiting data access
    
    **Solutions:**
    - Accept that Threads data may be less comprehensive
    - Supplement with Instagram profile data where available
    - Focus on core fields: username, bio, follower count
  </Accordion>
  
  <Accordion title="Scraping Rate Limits">
    **Possible Causes:**
    - Threads has different rate limiting than Instagram
    - Platform protecting new infrastructure
    
    **Solutions:**
    - Reduce scraping frequency
    - Implement longer delays between requests
    - Contact support for guidance on optimal scraping patterns
  </Accordion>
</AccordionGroup>

## Comparison with Instagram

### Key Differences

| Feature | Instagram | Threads |
|---------|-----------|----------|
| **User Base** | 2B+ users | Growing (smaller) |
| **Content Type** | Visual (photos/videos) | Text-based |
| **API Maturity** | Established | Developing |
| **Data Richness** | Comprehensive | Core fields |
| **Engagement Style** | Likes/comments on media | Text conversations |
| **Best For** | Visual brands, lifestyle | Thought leadership, discussions |
