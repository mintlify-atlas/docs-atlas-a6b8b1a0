## What is Click Creators Scraper Client?

Click Creators Scraper Client is an Instagram marketing automation platform that streamlines the process of scraping followers from influencer accounts, filtering profiles by gender, and distributing them to virtual assistants through Airtable.

The platform manages the complete workflow from scraping to assignment:
- **Multi-platform scraping** across Instagram, TikTok, X (Twitter), and Threads
- **Gender-based filtering** to identify male profiles using name detection
- **Campaign management** with daily batches of 14,400 profiles
- **Automated distribution** across 80 VA Airtable tables
- **Real-time tracking** with progress indicators and campaign history

## Tech Stack

The platform is built with modern technologies:

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, Radix UI
- **Backend**: Flask API (Python), Apify for scraping
- **Database**: Supabase (PostgreSQL) with Row Level Security
- **Integration**: Airtable API for VA distribution
- **Authentication**: Custom multi-factor authentication
